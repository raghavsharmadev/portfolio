## Development

To run locally:
```bash
ng serve
```
Then open [http://localhost:4200/](http://localhost:4200/) in your browser.

## Build for Production

```bash
ng build --base-href /portfolio/
```
This ensures correct asset paths for GitHub Pages deployment.
